# MiraiMart: Retail Intelligence & Demand Forecasting Platform

## Overview

MiraiMart is a retail analytics and demand forecasting solution designed to transform retail transaction data into actionable business insights. The project combines data analysis, forecasting, and interactive Power BI dashboards to help businesses monitor performance, optimize inventory, and support data-driven decision-making.

---

## Key Features

- Sales trend analysis
- Demand forecasting using Prophet
- Inventory performance monitoring
- Revenue and profit analysis
- Business KPI tracking
- Interactive Power BI dashboards
- Data preprocessing and exploratory analysis

---

## Project Structure

### File Description

| File | Purpose |
|--------|----------|
| dataset.xlsx | Retail sales dataset |
| holidays-code.py | Holiday data preparation for forecasting |
| main.ipynb | Data preprocessing, EDA, and forecasting |
| retail-analytics-dashboard-(Python).pbix | Dashboard generated using Python analytics |
| retail-analytics-dashboard-(SQL).pbix | Dashboard generated using SQL-based analysis |
